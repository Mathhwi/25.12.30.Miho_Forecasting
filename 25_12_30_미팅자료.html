2. 분석 방법론

누적지속강우량은 특정 시간 창(window) 동안의 총 누적 강우량입니다.

기존 강우 강도 분석에서는 누적강우량을 시간으로 나누었으나, 이번 분석에서는 나누지 않고 순수 누적량을 사용합니다.

[!] 핵심 변경점

기존: 최대 N시간 강우강도 = max(N시간 누적강우량) ÷ N

신규: 최대 N시간 누적강우량 = max(N시간 누적강우량) (나누지 않음)

이를 통해 절대적인 강우량 규모를 직접 비교할 수 있으며, 유역의 홍수 반응에 더 직접적인 연관성을 가질 것으로 기대됩니다.

용어 설명
  • p-value: 두 집단 간 차이가 우연에 의한 것일 확률. p < 0.05이면 통계적으로 유의미한 차이로 판단
  • Cohen's d: 효과 크기를 나타내는 지표. 절댓값 기준으로 |d| ≈ 0.2 (작은 효과), |d| ≈ 0.5 (중간 효과), |d| ≈ 0.8 (큰 효과)
  • Cohen's d 부호: 음수(-)는 비동시강우 평균이 동시강우 평균보다 낮음을 의미 (계산: 동시강우 - 비동시강우)
검정 결과 요약
  • 금천 관측소: 모든 window에서 유의미한 차이 없음 (p > 0.05)
  • 가덕 관측소: 3h, 6h, 9h, 12h에서 유의미한 차이 확인 (p < 0.05)
  • 12h 가덕: p = 0.011, |Cohen's d| = 0.40 (작은-중간 효과)

5. 결론

분석 결과 요약

1. 가덕 관측소의 누적강우량이 패턴 구분에 유의미

  • 금천 관측소: 동시/비동시강우 간 유의미한 차이 없음
  • 가덕 관측소: 3h, 6h, 9h, 12h에서 유의미한 차이 확인 (p < 0.05)

2. 비동시강우 시 가덕의 누적강우량이 상대적으로 낮음

  • 동시강우: 금천 57.8mm, 가덕 57.7mm (비율 1.00)
  • 비동시강우: 금천 53.6mm, 가덕 44.6mm (비율 1.20)

3. 12시간 누적강우량이 가장 효과적

  • p-value = 0.011, Cohen's d = -0.40 (중간 효과 크기)
  • 시간이 길어질수록 패턴 간 차이가 명확해짐
활용 방안
  • 모니터링 지표: 12시간 누적강우량의 금천/가덕 비율
  • 판단 기준: 비율 > 1.2일 경우 비동시강우 가능성 검토
  • 한계: 효과 크기가 중간 수준이므로 단독 판단보다는 보조 지표로 활용 권장
한계점 및 향후 과제

한계점

  • 효과 크기(Cohen's d = -0.40)가 중간 수준으로, 완벽한 구분은 어려움
  • 관측소가 2개로 제한되어 공간적 해상도가 낮음
  • 기상학적 원인 분석이 포함되지 않음

향후 과제

  • 금천/가덕 비율 1.2 기준의 예측 정확도 검증
  • 추가 관측소 데이터 확보 시 공간 분포 분석
  • 홍수 도달시간과의 연계 분석
